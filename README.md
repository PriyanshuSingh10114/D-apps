🏥 Decentralized Healthcare Application 📜🔗

A secure and transparent patient record management system powered by blockchain technology.

🌟 Features
🔒 Secure & Immutable Records – Patient data stored on the Ethereum blockchain.

👩‍⚕️ Patient & Doctor Access – Role-based permissions for data access.

💻 User-Friendly Dashboard – Built with ReactJS for seamless interaction.

🛠️ Smart Contract Backend – Developed in Solidity for secure transactions.

🔍 Tested & Reliable – Truffle framework for contract testing.

🛠️ Technologies Used
Category	Technology/Tool
Blockchain	Ethereum
Wallet Integration	Metamask
Local Blockchain	Ganache
Smart Contracts	Solidity
Frontend	ReactJS
Testing Framework	Truffle
Runtime	Node.js
🚀 Getting Started
Prerequisites
Node.js & npm installed

Metamask browser extension

Ganache (for local blockchain testing)

Installation
Clone the repository

bash
git clone https://github.com/your-repo/decentralized-healthcare.git
cd decentralized-healthcare
Install dependencies

bash
npm install
cd client && npm install
Run Ganache

Open Ganache and start a local Ethereum blockchain.

Deploy Smart Contracts

bash
truffle migrate --reset
Start the Frontend

bash
cd client && npm start
🧪 Testing
Run smart contract tests using Truffle:

bash
truffle test
